# Review Cycle — Claude Code Plugin

> [English version](README.md)

Автоматический цикл ревью PR для Claude Code. Запрашивает ревью, интеллектуально сортирует комментарии ревьюера, применяет исправления и повторяет до аппрува — затем squash-мержит.

## Установка

```bash
git clone https://github.com/axisrow/claude-code-review-cycle-skill.git
cp -r claude-code-review-cycle-skill/skills/review-cycle ~/.claude/skills/
```

### Требования

- [GitHub CLI](https://cli.github.com/) (`gh`), авторизованный в вашем аккаунте

## Использование

```
/review-cycle [pr-number]
```

Если номер PR не указан, скилл определяет его автоматически из текущей ветки.

## Как это работает

Скилл запускает автоматический цикл:

1. **Запрос ревью** — оставляет комментарий с просьбой сфокусироваться на критичных проблемах (баги, безопасность, логические ошибки)
2. **Ожидание ответа** — опрашивает PR 2–3 минуты
3. **Сортировка комментариев** — субагент читает реальный код и классифицирует каждый комментарий:

| Вердикт | Действие |
|---|---|
| `FIX` | Применить исправление (только после проверки утверждения по реальному коду) |
| `ALREADY_FIXED` | Ответить с указанием коммита, где это уже исправлено |
| `SKIP` | Ответить, что это косметическое замечание |
| `IRRELEVANT` | Ответить, что комментарий не относится к PR |
| `CONFLICTING` | Процитировать противоречащий комментарий, попросить уточнение |
| `HALLUCINATION` | Ответить с доказательствами из кодовой базы, опровергающими утверждение |

4. **Исправление** — применяет фиксы, запускает линтер и тесты
5. **Коммит и пуш** — conventional commit, возврат к шагу 1
6. **Завершение** — при аппруве squash-мержит и удаляет ветку

## Ключевые особенности

- Проверяет каждое утверждение ревьюера по реальной кодовой базе перед исправлением
- Обнаруживает галлюцинации: несуществующие функции, неверные номера строк, ложные конвенции
- Обрабатывает противоречия между циклами ревью
- Пропускает косметические замечания с вежливым объяснением
- Запускает линтер (`ruff`) и тесты (`pytest`) перед каждым пушем
